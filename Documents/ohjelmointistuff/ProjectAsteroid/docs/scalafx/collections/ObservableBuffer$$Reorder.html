scalafx.collections.ObservableBuffer

Reorder

case class Reorder(start: Int, end: Int, permutation: (Int) ⇒ Int) extends Change with Product with Serializable

Indicates a Reordering in an ObservableBuffer.

start

The start of the change interval.

end

The end of the change interval.

permutation

Function thst indicates the permutation that happened. The argument indicates the old index that contained the element prior to this change. Its return is the new index of the same element.
